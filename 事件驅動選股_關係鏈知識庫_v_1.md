# 📊 事件驅動選股 - 關係鏈知識庫 v1

---

## 🎯 核心概念

> 股價變動 = 資金 + 預期 + 事件傳導

本知識庫目標：
建立「消息 → 因子 → 產業 → 股票」的完整傳導鏈

---

## 🧠 五層架構（標準模型）

```
Event（事件）
→ Factor（影響因子）
→ Direction（正負向）
→ Industry（產業）
→ Stock（股票）
```

---

## 🏗️ 欄位設計（建議）

| 欄位 | 說明 |
|------|------|
| event | 事件名稱 |
| factor | 中介因子 |
| impact_sign | positive / negative |
| impact_strength | 1~5 |
| lag | immediate / short / medium / long |
| industry | 產業 |
| stocks | 股票列表 |

# 📚 事件關係鏈（核心庫）

---

## 1️⃣ 利率（Interest Rate）

### 升息

**Factor**
- 資金成本上升
- 折現率上升

**正向產業**
- 金融（中）

**負向產業**
- 成長股（高）
- REITs（高）
- 高股息（中）

---

### 降息

**Factor**
- 資金成本下降

**正向產業**
- 成長股（高）
- 房地產（中）

# 🚀 使用方式

### 計算模型

```
總分 = 基本面 + 籌碼 + 技術面 + 消息面
```

---

# ⚠️ 注意事項

1. 消息 ≠ 一定會漲
2. 市場看的是「預期差」
3. 同事件在不同環境影響不同

---

# 🧠 核心結論

> 市場不是對消息反應，而是對「消息的解讀」反應
